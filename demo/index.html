Revolve.js A bootstrap product viewer.

Usage

Include bootstrap.revolve.min.js in your document after bootstrap.

  • Anchor must point to an image in the sequence,
  • filenames must be suffixed with -numbers without leading zeros at the end i.e.: image-#.jpg,
  • frames option is required.

Via data attributes

Activate Revolve without writing JavaScript. Set data-toggle="revolve" on a controller element, like an anchor, along with a data-frames="#" and href="[image-sequence-path]" to open a specific revolve sequence in modal.

<a href="[image-sequence-path]" data-toggle="revolve" data-frames="24">...</a>

Via JavaScript

Attach bootstrap revolve to an elements with class .product with a single line of JavaScript: 

$('.product').revolve(options);

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-frame="".

Name type default description
frames int 24 Defines number of images in sequence.
template string - Contains default bootstrap modal markup without modal-header and modal-footer.

Methods

.revolve(options)

Activates your content with bootstrap revolve actions. Accepts an optional options object.

$('.product').revolve({
  frames: 30
});

.revolve('show')

Manually opens a revolve modal.

$('.product').revolve('show');

Events

All revolve events are fired at the anchor with bootstrap revolve attached (i.e. at the <a data-toggle="revolve">).

$('[data-toggle="revolve"]').on('loaded.bs.revolve', function (e) {
  // do something...
})
Event Type Description
loaded.bs.revolve This event is fired when revolve has pre-loaded all the images in sequence.
